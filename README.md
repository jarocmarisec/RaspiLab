## ¿Que hay en éste laboratorio?
Aquí documento mi evolución en el mundo de la programación de sistemas. Cada script es un "ladrillo" en la
construcción de un sistema de monitorización autónomo.
### El Centinela Obediente
*   **Función:** Monitorización térmica 24/7 de hardware.
*   **Estado:** 'Completado' | 'En Producción'
*   **Hitos:** Integración con API de Telegram y gestión como servicio de sistema (Systemd).
### El Notario Ciego
*   **Función:** Persistencia de datos y gestión de logs.
*   **Estado:** 'Testing'
*   **Hitos:** Manejo de archivos y marcas de tiempo (Timestamping).
